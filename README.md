# Web-Map-Interactions
###Description
This project displays a choropleth map of U.S. states shaded by the percentage of forest cover (timberland). Users can hover or click to explore per-state data. The data is sourced from Data.gov. 

###Packages Style Sheets and Data
Leaflet.js (for map creation and customization)
Leaflet.css and Google Fonts (stylesheets for the design of the user interface)
GeoJSON file (separate CSV + GeoJSON) containing state-level forest cover percentages.

###Data Source
The forest cover data (timberland only) comes from the U.S. Government open data portal (Data.gov)
